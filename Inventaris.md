# Sistem Inventaris Barang - Laravel

Project ini adalah aplikasi berbasis web untuk mengelola data inventaris barang, peminjaman, dan pengembalian.

# Fitur
- CRUD Data Barang
- Peminjaman Barang
- Pengembalian Barang
- Generate Laporan PDF
- Dashboard Statistik

# Teknologi
- PHP 8
- Laravel
- MySQL
- Bootstrap

# Cara Install
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` menjadi `.env`
4. Setup database
5. Jalankan `php artisan migrate`
6. Jalankan `php artisan serve`
